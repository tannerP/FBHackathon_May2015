#Running this website on your computer

##Mac
If you don't have brew, [click here](http://brew.sh).
###Install Node
```
brew install npm
```

###Install Node packages
```
cd path/to/project
npm install
```
###Install nodemon globally
```
npm install -g nodemon
```

###Start server
```
nodemon server.js
```
